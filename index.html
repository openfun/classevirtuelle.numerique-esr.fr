L'outil de classes virtuelles de l'enseignement supérieur

Dans le cadre du Plan de relance, cette solution de classe virtuelle s’appuie sur le logiciel open source BigBlueButton (BBB).
Elle est déployée à l’échelle nationale dans un cloud souverain opéré par la Direction du Numérique Éducatif (DNE) sous la responsabilité de France Université Numérique (FUN).

Les nouvelles fonctionnalités

En plus des fonctionnalités proposées par la solution BigBlueButton, bénéficiez de nouvelles options de configuration
Paramétrer sa classe une fois pour toute l'année
Planifier sa classe
Précharger des documents
Inviter des participants ou des modérateurs
Activer ou désactiver des options en amont
Saisir un texte de bienvenue

Vous enseignez ?

4 façons de créer vos classes virtuelles BigBlueButton !

Moodle

Utilisez BBB en LTI ou via le plugin dans le Moodle de votre établissement

Esup-Pod

Si votre établissement utilise Esup-Pod, lancez vos classes virtuelles depuis cette plateforme

Portail national des classes virtuelles

Connectez-vous avec vos identifiants universitaires pour lancer une classe virtuelle sans attendre

Plateformes FUN

Si vous créez des cours sur FUN MOOC, l'outil de classes virtuelles est préinstallé
Consulter le guide d'utilisation
